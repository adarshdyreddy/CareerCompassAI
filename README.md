# Career Compass AI

Career Compass AI is an AI-powered career guidance platform built using Python, Streamlit, and Google Gemini AI.

## Features
- Resume Analyzer
- Career Roadmap Generator
- Salary Insights
- Skill Gap Analyzer
- Interview Preparation
- Job Recommendations

## Technologies Used
- Python
- Streamlit
- Google Gemini AI
- GitHub
